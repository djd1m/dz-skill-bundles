# Storytelling Frameworks for Presentations

## Framework Selection Matrix

| Goal | Best Framework | Why |
|------|----------------|-----|
| Sell product/service | **AIDA** | Optimized for conversion |
| Change beliefs | **Hero's Journey** | Emotional transformation |
| Explain complex solution | **Problem-Solution** | Logical flow |
| Urgent action needed | **Monroe's Motivated** | Creates urgency |
| Technical decision makers | **Minto Pyramid** | Answer-first for busy execs |
| Investor pitch | **AIDA + Proof** | Credibility + conversion |

---

## 1. AIDA Framework

**Best for:** Product launches, sales presentations, investor pitches

```
A - ATTENTION (10-15% time)
    Hook with surprising fact, question, or bold claim
    
I - INTEREST (25-30% time)
    Build understanding of the problem/opportunity
    Show relevance to audience
    
D - DESIRE (35-40% time)
    Present solution
    Show proof (case studies, ROI, testimonials)
    Handle objections
    
A - ACTION (15-20% time)
    Clear CTA
    Remove friction
    Create urgency
```

### AIDA Slide Sequence

| Phase | Slides | Content |
|-------|--------|---------|
| Attention | 1-2 | Title with bold claim, Agenda |
| Interest | 3-5 | Problem amplification, Statistics, Trends |
| Desire | 6-12 | Solution overview, Features, Proof, TCO |
| Action | 13-15 | CTA options, Next steps, Contact |

### AIDA Transition Phrases

| From → To | Transition |
|-----------|------------|
| A → I | «Почему это важно именно сейчас?» |
| I → D | «Что с этим делать?» / «Есть выход» |
| D → A | «Как начать?» / «Следующий шаг» |

## 2. Problem-Solution Framework

**Best for:** Enterprise sales, technical presentations, complex products

```
PROBLEM (20-25% time)
    Current pain points
    Cost of inaction
    Regulatory pressure
    
ROOT CAUSE (10-15% time)
    Why existing solutions fail
    The real underlying issue
    
SOLUTION (30-35% time)
    Your approach
    Key differentiators
    How it works
    
PROOF (15-20% time)
    Case studies
    TCO comparison
    Testimonials
    
CTA (10-15% time)
    Clear next steps
    Risk reduction (pilot, trial)
```

### Problem-Solution Slide Sequence

| Phase | Slides | Content |
|-------|--------|---------|
| Problem | 1-4 | Hook, Pain points, Statistics, Timeline |
| Root Cause | 5-6 | False dilemma, Why alternatives fail |
| Solution | 7-12 | Pillars, Product details, Architecture |
| Proof | 13-16 | TCO, Case studies, Comparison |
| CTA | 17-20 | Options, Checklist, Why us, Contact |

### Problem-Solution Emotional Arc

```
TENSION         ████████████████████░░░░░░░░░░░░░░░░░░░░  (Problem)
                ████████████████████████░░░░░░░░░░░░░░░░  (Root Cause)
RELEASE         ░░░░░░░░████████████████████████████████  (Solution)
CONFIDENCE      ░░░░░░░░░░░░░░░░████████████████████████  (Proof)
ACTION          ░░░░░░░░░░░░░░░░░░░░░░░░████████████████  (CTA)
```

## 3. Hero's Journey (Simplified)

**Best for:** Keynotes, vision presentations, culture change

```
ORDINARY WORLD (10% time)
    Current state, familiar territory
    
CALL TO ADVENTURE (15% time)
    Challenge appears, opportunity emerges
    
REFUSAL / OBSTACLES (15% time)
    Why it's hard, what's stopping us
    
MENTOR / SOLUTION (20% time)
    Guide appears (your product/service)
    
TRANSFORMATION (25% time)
    The journey, implementation, change
    
NEW WORLD (15% time)
    Results, new reality, future vision
```

### Hero's Journey for Business

| Story Element | Business Translation |
|---------------|---------------------|
| Hero | Customer / Audience |
| Ordinary World | Current processes, status quo |
| Call to Adventure | Market change, new regulation, opportunity |
| Refusal | Objections, fears, "we tried before" |
| Mentor | Your company / solution |
| Transformation | Implementation journey |
| New World | After state, ROI, success |

## 4. Monroe's Motivated Sequence

**Best for:** Persuasive speeches, urgent calls to action

```
ATTENTION (5-10% time)
    Shocking fact or story
    
NEED (20-25% time)
    Problem that affects audience directly
    
SATISFACTION (30-35% time)
    Solution that addresses the need
    
VISUALIZATION (20-25% time)
    Paint picture of success (and failure without action)
    
ACTION (10-15% time)
    Specific steps to take NOW
```

### Visualization Techniques

**Positive Visualization:**
> «Представьте: через год ваша команда...»

**Negative Visualization:**
> «А теперь представьте, что вы не предпринимаете никаких действий...»

**Contrast:**
> «Сравните: сегодня vs через 12 месяцев с [solution]»

## 5. Minto Pyramid

**Best for:** Executive briefings, technical decisions, busy audiences

```
ANSWER FIRST (5-10% time)
    State conclusion upfront
    
KEY ARGUMENTS (40-50% time)
    3-5 supporting points (MECE)
    
DETAILS (30-40% time)
    Evidence for each argument
    
IMPLICATIONS (10-15% time)
    What this means, next steps
```

### Minto for Presentations

| Slide | Content |
|-------|---------|
| 1 | **Answer:** «Рекомендуем [X] потому что [Y]» |
| 2-4 | **Argument 1** + evidence |
| 5-7 | **Argument 2** + evidence |
| 8-10 | **Argument 3** + evidence |
| 11-12 | **Implications** + next steps |

## Emotional Beats

### High-Tension Moments (use sparingly)

- Surprising statistics
- "What if" scenarios  
- Competitor failures
- Regulatory deadlines
- Cost of inaction

### Relief/Resolution Moments

- Solution reveal
- Success stories
- Demo/proof
- Clear path forward

### Connection Moments

- Rhetorical questions
- "Raise your hand if..."
- Shared frustrations
- Industry jokes/references

## Framework Combinations

### Enterprise Sales (Problem-Solution + AIDA)

```
[Problem-Solution structure]
 + [AIDA emotional arc]
 = Problem as Attention
   Root Cause builds Interest
   Solution creates Desire
   Proof reinforces Desire
   CTA drives Action
```

### Investor Pitch (Minto + AIDA)

```
Slide 1: Answer (what we do, traction)
Slides 2-4: Market (AIDA-Attention)
Slides 5-7: Product (AIDA-Interest)
Slides 8-10: Traction (AIDA-Desire)
Slides 11-12: Ask (AIDA-Action)
```

---

## Anti-Patterns

❌ **Starting with history** — Nobody cares about your founding story
❌ **Burying the lead** — Don't save the main point for slide 15
❌ **All logic, no emotion** — Even B2B buyers are human
❌ **Tension without release** — Don't leave audience anxious
❌ **Solution without proof** — Claims need evidence
❌ **CTA without path** — "Contact us" is too vague
